# FalconViewCarRentals Data Layer

This directory contains the database and data-related scripts for the backend.

- `sqlite/`: Contains the SQLite database file (`falconview.db`).
- `alembic/`: Alembic migrations configuration and scripts.
- `seeds/`: Python scripts to extract data from React frontend and seed SQLite database.
- `scripts/`: Useful scripts for backups and maintenance.
